# WndDesign

A C++ GUI library

## Quick Start

*for Windows 10/11 only*

* Install Visual Studio 2022, with **Desktop development with C++** selected.

* Clone `WndDesign` source code.

* Open `WndDesign.sln`, set `Test` as startup project, build, and run.
